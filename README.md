# hosted-webfiles

Files that I'm hosting for websites, including Neocities sites and VS Code assets.

## Directory Structure

- **`fonts/`** - Font files (TTF, WOFF, WOFF2, OTF, etc.)
- **`images/`** - Image files (PNG, JPG, GIF, SVG, etc.)
- **`videos/`** - Video files (MP4, WEBM, etc.)
- **`neocities/`** - Assets specifically for Neocities websites
- **`vscode/`** - Assets for VS Code extensions and themes

## Usage

Files in this repository are hosted on GitHub and can be accessed via:
- GitHub Pages (if enabled)
- Raw GitHub URLs: `https://raw.githubusercontent.com/eps-k/hosted-webfiles/main/<path-to-file>`
- jsDelivr CDN: `https://cdn.jsdelivr.net/gh/eps-k/hosted-webfiles@main/<path-to-file>`

## Contributing

To add new files:
1. Place them in the appropriate directory
2. Commit and push to the repository
3. Use the file URLs in your websites or projects
